# AirBnB Dashboard - Tableau Project

This project analyzes AirBnB data using Tableau.

**Data Sources:**
- Listings
- Calendar
- Reviews

**Key Visualizations:**
- Price by Zipcode
- Price Per Zipcode
- Revenue by Week
- Average Price Per Bedroom
- Distinct Count of Bedroom Listings

**Tools Used:**
- Tableau
- Inner Joins
- Aggregations
- Filters and Formatting
